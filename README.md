# TLMC Helper

TLMC Helper is a small workflow for scanning music tags, reviewing album
metadata in a web app, and writing cleaned tags back to audio files.

## Usage

1. Arrange your music library using the expected TLMC folder layout.
2. Run `scan-albums` in the library root to create `metadata.json`.
3. Open the web app.
4. Import `metadata.json`.
5. Optionally import existing `structured.json` and `rewriting.json`.
6. Review and edit album metadata and rewrite rules.
7. Click `Sync now`.
8. Download `update-metadata.json`.
9. Run `apply-tags` in the library root to write the updates.

Only circles marked as audited in the Rewrite Rules view are included in
`update-metadata.json`. This keeps unfinished circles out of the patch file.

Suggested workflow after importing `metadata.json`:

1. Check the audit log. In particular, look at `missing_info`,
   `inconsistent_date`, and `disc_classification`; these suggest where to look
   in the Album Metadata view.
2. Check the Rewrite Rules view. Look at the artist, album-artist, and genre
   name lists, then inspect the rewrite rules. Change or remove incorrect
   autogenerated rules, add any further rewrite rules you need, click
   `Sync now`, and review the name list again. If a name looks weird because
   the source metadata is wrong rather than something rewrite rules should fix,
   double-click the name to jump to the first matching Album Metadata field.
   Repeat until the circle looks right.
3. After making sure one circle is good, click its `Audited` checkbox.
4. After dealing with all circles, check the `$all` circle to see whether names
   are consistent across circles.

The web app stores your imported data and edits locally in your browser, so
closing the tab should not lose your current work. You can still download
`structured.json` and `rewriting.json` when you want backups or files to reuse
later.

If you need to split albums from archive/CUE sources first, run `split-album`
before `scan-albums`.

## Binaries

Windows users can download the latest `split-album.exe`, `scan-albums.exe`, and
`apply-tags.exe` from the GitHub Actions page:

https://github.com/pca006132/tlmc-helper/actions

To build a release version locally:

```bash
cargo build --release
```

The binaries are written to `target/release/`.
